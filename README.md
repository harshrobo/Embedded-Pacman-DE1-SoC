# Embedded Pac-Man Game on DE1-SoC

An embedded implementation of the classic Pac-Man game developed on the DE1-SoC platform using the ARM Cortex-A9 processor and LT24 LCD display.

---

## Features

- Intro video animation
- Maze generation
- Pacman movement
- Collision detection
- Ghost AI
- Background music
- Touchscreen support
- LT24 LCD graphics
- Real-time gameplay

---

## Hardware

- DE1-SoC Development Board
- ARM Cortex-A9
- LT24 LCD Display
- WM8731 Audio Codec
- FPGA GPIO
- HPS I2C
- Seven Segment Display

## Project Modules

| Module | Description |
|--------|-------------|
| Audio | WM8731 audio playback |
| Video | Intro animation |
| Maze | Tile-based maze generation |
| Pacman | Movement and collisions |
| Ghost AI | BFS-based ghost movement |
| Touchscreen | ADC and GPIO driver |
